Closed-phase snapshots of working artifacts. Used to clear the active `_working` root after an iteration (e.g. S03) without losing history.

- **s03_closeout/** — Artifacts archived at S03 close (reports, issue/PR drafts, research drafts, docs_hygiene, protocol_research). See `s03_closeout/S03_closeout_classification.md` for what was moved.

Do not use this tree for active work; keep new work in the main `_working` folders (research, hw_tests, issues, pr, etc.).
