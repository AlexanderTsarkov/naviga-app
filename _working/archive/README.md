# Working archive

Closed iteration and phase materials live here.

**Convention:** `_working/archive/iterations/<iteration-tag>/...`  
Use the iteration tag from the first line of `_working/ITERATION.md` (e.g. `S05__2026-03__Android_Mobile_Foundation_on_BLE.v1`).

**Older archives:** Some prior phase archives may still live under `_archive/iterations/` at repo root. Treat this folder (`_working/archive/`) as the standard going forward. Migrate older materials only in dedicated housekeeping passes.
